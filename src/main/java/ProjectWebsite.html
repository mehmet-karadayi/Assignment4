Lyrics

  • Hash-map was used to calculate the word frequencies. The key and value feature of the Hash-map enabled me to lockdown each word and count the number of times said word appeared in the lyrics.
  • The visualization technique of choice was a word-cloud. I used IntelliJ IDE to import the Kumo library word cloud creator through the Maven open source library to create the word cloud above. The word frequency list was created using a java program from the previous assignment.
  • If I had more time (more knowledge of HTML/CSS) the overall look of the site and its content would look more official. This is my first time writing code to create a website. However, writing code to create something visual, website/word-cloud, is entertaining.
  • Saturday Night's Alright (For Fighting) is one of my favorite songs. It wasn't surprising to find that the word "Saturday" is the most repeated word in the song. This song was released as a single and radio's feared that that it might incite violence. In the year that it was released, 1973, this song was at the 7th spot of UK Singles Chart and 12th spot on the US Billboard Hot 100.
